---
title: "Dome API - Changelog"
description: "Heres a list of changes and additions to the API overtime to track new features and upgrades"
---

## January 7th, 2026
- Made orderbook time filters optional - will now return the latest orderbook data by default

## January 6th, 2026
- Added fuzzy search support to both [Kalshi Markets API](https://docs.domeapi.io/api-reference/endpoint/get-kalshi-markets) and [Polymarket Markets API](https://docs.domeapi.io/api-reference/endpoint/get-markets)

## December 24th, 2025
- Updates to SDK for Typescript and Python

## December 20th, 2025
- Added [college basketball](https://docs.domeapi.io/api-reference/endpoint/get-matching-markets-sports-sport?playground=open) to matching markets

## November 29th, 2025
- Beta for Polymarket Order Router launched, email kunal@domeapi.com for access

## November 19th, 2025
- Added wallet metrics to the [/wallet API](https://docs.domeapi.io/api-reference/endpoint/get-wallet) to showcase volume and merges, splits, conversions information
- Performance updates for orders API

## November 10th, 2025
- Added market slug and condition ID filters to [websockets](https://docs.domeapi.io/websockets/subscribe)
- Added [update subscription](https://docs.domeapi.io/websockets/update) action to websockets for modifying existing subscriptions without re-subscribing

## November 9th, 2025
- Added [websocket support](https://docs.domeapi.io/websockets), docs, and SDKs

## November 8th, 2025
- Changed the maxium limit for orderbooks for both Kalshi and Polymarket from 500 -> 200. Will default to 200 limit is higher.

## October 31st, 2025
- Added the [Kalshi Markets API](https://docs.domeapi.io/api-reference/endpoint/get-kalshi-markets) to search and filter Kalshi markets
- Added the [Kalshi Orderbook History API](https://docs.domeapi.io/api-reference/endpoint/get-kalshi-orderbook-history) with support for historical orderbook snapshots

## October 30th, 2025
- Added a status filter (`open` or `closed`) to the [Polymarket Markets API](https://docs.domeapi.io/api-reference/endpoint/get-markets)
- Added a min_volume filter to the [Polymarket Markets API](https://docs.domeapi.io/api-reference/endpoint/get-markets)

## October 26th, 2025
- Change the max limit from 1,000 items to 500 on the [Polymarket orderbook API](https://docs.domeapi.io/api-reference/endpoint/get-orderbook-history)
- Added `pagination_key` field to [Polymarket orderbook API](https://docs.domeapi.io/api-reference/endpoint/get-orderbook-history) (will keep `paginationKey` for backwards compatiability)
- Added a filter to the [markets API](https://docs.domeapi.io/api-reference/endpoint/get-markets) to filter on `event_slug(s)`
- Added the Polymarket [Activity API](https://docs.domeapi.io/api-reference/endpoint/get-activity)
